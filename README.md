### 🚀 Projetos em Destaque

#### [🌌 Celestella](https://github.com/devitorhugocalca/celestella)
**Onde as Redes Sociais encontram o RPG.** Uma aplicação web Full Stack construída do zero (Vanilla JS + Node.js/MongoDB) que injeta mecânicas de jogos Arcade e RPG diretamente no feed do usuário, com comunicação em tempo real via WebSockets.

#### [☕ Caffé no Quintal](https://www.caffenoquintal.com.br/)
**Presença Digital & Institucional.** Desenvolvimento da plataforma web para uma cafeteria localizada em Moema, São Paulo. Foco em performance, design acolhedor e otimização para atração de clientes locais.

#### [🥐 Bossa Bakery](https://www.bossabakery.com.br/)
**Identidade Visual e Experiência do Usuário.** Site institucional desenvolvido para destacar os produtos da padaria. Focado em navegação intuitiva e design totalmente responsivo para valorizar o catálogo e a conversão de clientes.

#### [🥩 O Brazeiro - Cardápio Digital](https://www.obrazeiro.com.br/new-menu/)
**Solução em Menu Digital.** Estruturação e desenvolvimento de uma interface moderna para o cardápio do restaurante, projetado com foco estrito na usabilidade mobile, aliando os princípios de desenvolvimento web às estratégias de mercado gastronômico.
